# Portfolio - Frontend Client

This is the frontend client for the portfolio website. It is built as an interactive React Single Page Application (SPA) powered by Vite.

## Tech Stack
- **Framework**: React 19
- **Bundler / Dev Server**: Vite 8
- **Styling**: Tailwind CSS v4 & custom inline CSS for visual layout control
- **Animations**: Framer Motion
- **API Client**: Axios

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run in Development Mode**:
   ```bash
   npm run dev
   ```
   *The client app runs locally at `http://localhost:5173`.*

3. **Build for Production**:
   ```bash
   npm run build
   ```
   *The built static files are compiled into the `dist/` directory.*
